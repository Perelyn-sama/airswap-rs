# airswap-rs

## Quickstart

Add this to your Cargo.toml:

```toml
[dependencies]
airswap-rs = { git = "https://github.com/Perelyn-sama/airswap-rs" }
```

And this to your code:

```rust
use airswap::prelude::*;
```
